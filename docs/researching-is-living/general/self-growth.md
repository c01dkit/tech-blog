# 个人成长

成长是不断打磨自己的过程。相比被动式（被鞭策）的改变，在信息爆炸的背景下，要主动探索自己。

## 锻炼思维

**批判性。**之前看小说提到过一种锤炼思维的方法，就是先写下自己认为的一些“绝对正确”的固有观念，然后从多个角度来进行否定。类似地，也可以在工作记录中批判性地思考认知漏洞。即，不只是记录做了什么，还要进一步思考：这些成果中哪些可能是错误的？是出发点还是路径错误？领域大牛会这么做吗？会不会存在边界条件与反例？

**知识网络。**将杂乱的知识整合成知识图谱，在规模比较大的时候可以鸟瞰全局，分析出热点和趋势。

## 勤奋与懒惰

不要用战术上的勤奋来掩盖战略上的懒惰，忙来忙去像是无头苍蝇。虽然很痛苦，但是还要去想想：

1. 明确自己的核心定位。目前在做的事情与未来三到五年的成长有没有关系？哪些事情高收益？哪些事情低收益？
2. 对要做的任务进行归类划分。即常说的四象限法，从相关性和难度上可以分为：高关联高难度的投入主要精力，高关联低难度设计标准化方法，低关联高难度请求协助或放弃，低关联低难度立即放弃。
3. 主动思考破局之路。不要停留在表面的功夫，首先定位领域最重要的知识点、技能树，关注领域大牛动态；用输出（写文章）来倒逼输入（学习）和处理（思考）；主动关联多个领域，它山之石可以攻玉。
4. 保持灵活性、多样性。善于合理分配时间，主要精力用于投入专业素养与技能提升，次要精力进行跨领域学习。
